autodoxygenfile Directive Example¶

For more details and directive documentation please see doxygenfile Directive, which is very similar to this directive.

Working Example¶

This should work:

.. autodoxygenfile:: auto_class.h
   :project: auto

With the following config value:

breathe_projects_source = {
     "auto" : ( "../examples/specific", ["auto_class.h"] )
     }

It produces this output:

class AutoClassTest¶
#include <auto_class.h>

class outside of namespace

Private Functions

inline void member()¶

non-namespaced class function

inline void anotherMember()¶

non-namespaced class other function
