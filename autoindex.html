autodoxygenindex Directive Example¶

Working Example¶

This should work:

.. autodoxygenindex::
   :project: auto

With the following config value:

breathe_projects_source = {
     "auto" : ( "../examples/specific", ["auto_function.h", "auto_class.h"] )
     }
